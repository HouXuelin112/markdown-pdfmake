# markdown-pdfmake

将 Markdown 文本转换为 pdfmake 文档定义的工具库，用于生成 PDF 文档。

## 功能特性

- 支持基本 Markdown 语法：标题、段落、列表、表格、图片、水平线、引用块
- 支持代码块语法高亮
- 支持内联样式：粗体、斜体、链接等
- 类型安全：使用 TypeScript 编写，提供完整的类型定义
- 可扩展：可通过自定义 context 进行配置

## 安装

```bash
# 使用 npm
npm install markdown-pdfmake

# 使用 yarn
yarn add markdown-pdfmake

# 使用 pnpm
pnpm add markdown-pdfmake
```

## 依赖

- [markdown-it](https://github.com/markdown-it/markdown-it) - Markdown 解析器
- [pdfmake](https://github.com/bpampuch/pdfmake) - PDF 生成库
